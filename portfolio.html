Sports Matchmaking Application (Capstone)

Feb 2025 - Present

Spearheading the development of a responsive web application designed to connect sports enthusiasts and facilitate the organization of local sports matches. This project involves leading a team of four developers, ensuring effective collaboration and timely delivery of features.

The front-end is built using Bootstrap Studio for rapid prototyping and JavaScript for dynamic interactions, ensuring a seamless user experience across devices. On the back-end, the application leverages Node.js to create RESTful APIs, enabling secure and efficient communication between the client and server. Data is stored and managed using PostgreSQL, ensuring scalability and reliability for user profiles, match details, and messaging features.

This project emphasizes clean code practices, version control with Git, and regular code reviews to maintain high-quality standards. It also incorporates Agile methodologies, with weekly sprints and team retrospectives to ensure continuous improvement.

Online Course/Task Tracker

Sep 2024 - Oct 2024

Built a full-stack web application to help students track coursework and deadlines. This project was inspired by a 2019 Medium article that highlighted the power and flexibility of the MERN stack (MongoDB, Express.js, React.js, Node.js) for building modern web applications.

The front-end was developed using React.js for dynamic and responsive UI components, while the back-end utilized Node.js and Express.js to create RESTful APIs. Data was stored in MongoDB, ensuring efficient and scalable storage for user tasks and deadlines.

Docker was a critical part of this project, enabling containerized deployment to ensure consistency across development, testing, and production environments. This allowed the application to run seamlessly across different systems without compatibility issues.

This solo project followed Agile development principles and incorporated unit and integration testing using tools like Postman and Studio 3T to uphold high-quality standards throughout backend development.

Game Development (Capstone)

Aug 2021 - May 2022

Created an open-world 3D game using Unreal Engine and Blender for modeling. Integrated assets via the Epic Store and implemented interactive mechanics to enhance gameplay.

Key features include:

  • Realistic physics and animations for an immersive experience.
  • Custom set behavior for non-player characters (NPCs) to simulate realistic behavior. This includes mobs, animals, boss characters, etc. Each NPC has unique interactions and unique set of behaviors, attack patterns, and animations based on the game environment.
  • Ability to interact with ingame items/elements.
  • Dynamic weather and day-night cycle to enhance realism.
This project emphasized iterative development and continuous improvement, with regular playtesting sessions to ensure a high-quality user experience.
